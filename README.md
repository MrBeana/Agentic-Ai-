# 👋 About This Repository

Welcome to my AI portfolio.

This repository contains practical **Generative AI**, **Retrieval-Augmented Generation (RAG)**, and **Agentic AI** projects built using modern LLM technologies.

The goal of these projects is to explore how intelligent systems can retrieve knowledge, collaborate through multiple AI agents, process real-world documents, and generate accurate, context-aware responses.

Each project is designed as a production-style application with a clean Gradio interface, modular architecture, and extensible codebase.

### Features

* FAISS vector search

* Context-aware question answering

## 🔮 Project 4 — AI Jathakam Reader

A multilingual AI assistant capable of reading and interpreting Indian astrology birth charts (Jathakam).

The system extracts information from PDFs and images using Vision models, generates structured interpretations, and enables conversational interactions with the chart.

### Features

* OCR using GPT-4 Vision
* Structured chart extraction
* Astrology interpretation
* South Indian chart visualization
* Malayalam, Sanskrit & English support
* RAG-powered chat
* SVG horoscope rendering

**Tech Stack**

* GPT-4o Vision
* FAISS
* Python
* Gradio
* OCR
* SVG Visualization
